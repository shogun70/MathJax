MathJax Test and Sample pages

Thankyou for installing MathJax - "Beautiful math in all browsers". The following pages can help confirm that MathJax is installed correctly, and identify issues with the installation or configuration in a problem page.

  • test.html
    A basic MathJax enabled page. Use this to confirm that MathJax has been installed correctly, together with standard MathJax web-fonts.
    This page should be checked in at least Internet Explorer, Firefox (or Safari) and Chrome, because different web-font formats are supported by the different browsers.
  • test-images.html
    A basic MathJax enabled page that uses image-fonts. Image-fonts are the fallback option provided by MathJax for browsers that don't support web-fonts. Use this to confirm that the image-fonts have been installed correctly.
  • checklist.html
    Checks a cross-section of MathJax files highlighting success (or failure) and performance issues.
    This can give an indication of why a MathJax installation seems to fail or be slow.

    • Also, here is the code for a javascript bookmarklet which creates a diagnostics window for a web-page. This code requires the MathJaxToolsDir to be configured and to be in the same domain as the page under test.

    var MathJaxToolsDir = "/mathjax/tools"; var script = document.createElement("script"); script.src = MathJaxToolsDir + "/diagnostix.js"; document.getElementsByTagName("head")[0].appendChild(script); void(0);

    There are also several sample pages that demonstrate how to configure and / or program with MathJax.

  • sample.html
    A page containing math in TeX format
  • sample-tex.html
    A basic page containing math in TeX format
  • sample-mml.html
    A basic page containing math in MathML format. The appearance should be the same as sample-tex.html
  • sample-tex2mml.html
    A basic page containing math in TeX format. If the browser has native support for MathML then the rendering is delegated to the browser. Otherwise the appearance should be the same as sample-tex.html
  • sample-signals.html
    A basic page demonstrating how to use MathJax's signal mechanism to find out about what MathJax is doing, and to hook into various events as they occur.
  • sample-dynamic.html
    A simple demonstraton of programming with MathJax to update math on the page in response to user input.
