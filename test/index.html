Beautiful math in all browsers

Thankyou for installing MathJax. This page lists various resources that can help test your installation and MathJax enabled pages.

Installation Tests

The following pages can help confirm that MathJax is installed correctly, and identify issues with the installation or configuration in a problem page.

  • install.html
    A basic MathJax enabled page. Use this to confirm that MathJax has been installed correctly, together with standard MathJax web-fonts.
    Beause different browsers support different web-font formats, this page should be checked in at least the following browsers
    • Internet Explorer
    • Safari
    • Firefox
    • Chrome
  • install-images.html
    A basic MathJax enabled page that uses image-fonts. Image-fonts are the fallback option provided by MathJax for browsers that don't support web-fonts. Use this to confirm that the image-fonts have been installed correctly.
  • install-diag.html
    Checks a cross-section of MathJax files highlighting success (or failure) and performance issues.
    This can give an indication of why a MathJax installation seems to fail or be slow.

    • Page Tests

    The following tools perform simple diagnostics and performance analysis of a MathJax enabled page.

  • page-diag.html
    Provides a few basic checks of a MathJax enabled page.
    Also takes a comprehensive snapshot of browser, window and MathJax configuration which may be posted to a support group for analysis.
    Provides basic timing charts for a MathJax enabled page.
    These can give an indication of where a page seems to fail or be slow. This page cannot run the checks if this directory is in a different domain to the page under test.
  • page-diag.js
    This script can be added to the head of a MathJax enabled page to import the testing functionality provided by page-diag.html.

    • Sample Pages

    There are also several sample pages that demonstrate how to configure and / or program with MathJax.

  • sample.html
    A page containing math in TeX format
  • sample-tex.html
    A basic page containing math in TeX format
  • sample-mml.html
    A basic page containing math in MathML format. The appearance should be the same as sample-tex.html
  • sample-tex2mml.html
    A basic page containing math in TeX format. If the browser has native support for MathML then the rendering is delegated to the browser. Otherwise the appearance should be the same as sample-tex.html
  • sample-signals.html
    A basic page demonstrating how to use MathJax's signal mechanism to find out about what MathJax is doing, and to hook into various events as they occur.
  • sample-dynamic.html
    A simple demonstraton of programming with MathJax to update math on the page in response to user input.
